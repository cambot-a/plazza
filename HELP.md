# PLAZZA
Concurrent pizzeria simulation using processes, threads, IPC, thread pools, and load balancing.

## Usage
```bash
./plazza <multiplier> <cooks> <reload_time_ms>
```

Example order:
```txt
regina XXL x2; fantasia M x1
```

Commands:
- `status` | print the status of the program
- `exit` | exit the program
